# optrn_simulator
GUI for Opentrons Simulator

# packages the app is using
- tkinter
- request
- pyinstaller

# how to run the app
```zsh
git clone https://github.com/koji/optrn_simulator.git
cd optrn_simulator
pip install -r requirements.txt
python app.py
```

# create the app
```zsh
pyinstaller --onefile app.py
```
